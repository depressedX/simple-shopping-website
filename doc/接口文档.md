# 科研项目购物网站接口文档
## 全局
+ **API地址:** http://host:port/api/
+ **前端路由** 访问所有路径均返回/index.html
+ **返回信息格式说明**
  ```json
    {
      "status":200,
      "message":"OK",
      "data":null
    }
    ```
  + 成功  
  status 200  
  message OK
  + 失败  
  status 4**  
  message 具体错误描述
## 具体请求
### 商品相关
#### 获取商品列表
**url**:/goods  
**method**:GET
  
**desc**:  
page 对商品分页 默认页数 0  
rows 每页的数量 默认12
  
request:  
```json
{
"page":0,
"rows":12 
}
```

response:  
```json
{
"status":200,
"message":"OK",
"data":[
{
"id":0,
"name":"string",
"price":30.00 
}
]
}
```

#### 获取单个商品
**url**:/goods/:id  
**method**:GET
  
**desc**:  
  
request:  
```json
{
"id":12 
}
```

response:  
```json
{
"status":200,
"message":"OK",
"data":{
"id":0,
"name":"string",
"price":30.00 
}
}
```

#### 新增商品
**url**:/goods  
**method**:GET
  
**desc**:  
返回新增商品的ID
  
request:  
```json
{
"name":"string",
"price":12.00
}
```

response:  
```json
{
"status":200,
"message":"OK",
"data":{
"id":12
}
}
```

#### 修改商品信息
**url**:/goods/:id  
**method**:POST
  
**desc**:  
除ID都可修改(暂时是这样的)
  
request:  
```json
{
"name":"string"
}
```

response:  
```json
{
"status":200,
"message":"OK"
}
```
