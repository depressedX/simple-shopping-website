# 科研项目购物网站接口文档
## 全局
+ **API地址:** http://host:port/api/
+ **前端路由** 访问所有路径均返回/index.html
+ **返回信息格式说明**
  ```json
    {
      "status":200,
      "message":"OK",
      "data":null
    }
    ```
  + 成功  
  status 200  
  message OK
  + 失败  
  status 4**  
  message 具体错误描述
+ /:param 表示*动态路径参数*  
------------------------------
## 具体请求
### 商品相关

2017//11//14:新增added字段  表示已经加入了购物车  
     和ID一样不可以更改的   在添加物品\删除物品的时候在后台改

#### 获取商品数量
**url**:/item/total  
**method**:GET
  
**request**:  
```json
{
}
```
**response**:  
```json
{
"status":200,
"message":"OK",
"data":10
}
```

#### 获取商品列表
**url**:/items  
**method**:GET
  
**desc**:  
page 对商品分页 默认页数 1  
rows 每页的数量 默认12
  
**request**:  
```json
{
    "page":1,
    "rows":12 
}
```

**response**:  
```json
{
"status":200,
"message":"OK",
"data":[
    {
        "itemId":0,
        "name":"string",
        "price":30.00 ,
        "imgSrc":"src",
        "added":false
    }
]
}
```

#### 获取单个商品
**url**:/item/:itemId  
**method**:GET
  
**desc**:  
  
**request**:  
```json
{
    "itemId":12 
}
```

**response**:  
```json
{
"status":200,
"message":"OK",
"data":{
    "itemId":0,
    "name":"string",
    "price":30.00 ,
    "imgSrc":"src",
    "added":false
}
}
```

#### 新增商品
**url**:/item/add  
**method**:POST
  
**desc**:  
返回新增商品的ID
  
**request**:  
```json
{
    "name":"string",
    "price":12.00,
    "imgSrc":"图片地址"
}
```

**response**:  
```json
{
"status":200,
"message":"OK",
"data":{
    "itemId":12
}
}
```

}

### 购物车相关

**desc**  
cartId:购物车物品条目ID  
itemId:商品ID  
name:商品名称  
price:商品单价  
num:商品数量
注:商品数量为0后自动删除该条目  

**一条购物车条目指一定数量的某一个商品**

#### 获取购物车条目数量
**url**:/cart/total  
**method**:GET
  
**desc**:  
比如 棒球帽*3 上衣*1 则条目数量为2
  
**request**:  
```json
{
}
```
**response**:  
```json
{
"status":200,
"message":"OK",
"data":2
}
```

#### 获取购物车物品
**url**:/carts  
**method**:GET
  
**desc**:  
  
**request**:  
```json
{
}
```
**response**:  
```json
{
"status":200,
"message":"OK",
"data":[
    {
        "cartId":0,
        "itemId":12,
        "name":"string",
        "price":30.00,
        "num":2
    }
]
}
```

#### 向购物车添加物品
**url**:/cart/add  
**method**:POST
  
**desc**:  
如果添加的物品(item)已经存在于购物车,则物品数目增加num
如果不存在  则新增购物车条目
  
**request**:  
```json
{
    "itemId":12,
    "num":2
}
```

**response**:  
```json
{
"status":200,
"message":"OK",
"data":{
    "cartId":12
}
}
```
```json
{
"status":401,
"message":"超出购物车最大商品数",
"data":{
    "cartId":12
}
}
```

#### 修改购物车条目信息
**url**:/cart/update/:cartId  
**method**:POST
  
**desc**:  
目前只能修改数量
  
**request**:  
```json
{
    "num":2
}
```
**response**:  
```json
{
    "status":200,
    "message":"OK"
}
```

#### 批量删除购物车条目
**url**:/carts/delete  
**method**:POST
  
**desc**:  
批量删除  cartIdList为要删除的条目ID  
用英文,分隔
  
**request**:  
```json
{
    "cartIdList":"1,3,5,7"
}
```
**response**:  
```json
{
    "status":200,
    "message":"OK"
}
```
